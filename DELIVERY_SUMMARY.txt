## 📊 Technical Specifications

### **Input Specification**
- Format: JPG, PNG (any size supported)
- Processing: Auto-resized to 224×224
- Normalization: [-1, 1] range (MobileNetV3 optimized)
- Color: RGB (auto-converted from BGR)

### **Model Architecture**
```
Input (224×224×3)
    ↓
MobileNetV3Small backbone (frozen ImageNet weights)
    ↓
Global Average Pooling
    ↓
Dense(256) + Dropout(0.3) → ReLU
    ↓
Dense(128) + Dropout(0.2) → ReLU
    ↓
Output Layer:
  - Stage 1: Dense(1) + Sigmoid → Binary
  - Stage 2: Dense(4) + Softmax → Multi-class
```

### **Training Configuration**
- Optimizer: Adam (lr=0.0001)
- Loss: Binary Crossentropy (Stage 1), Categorical Crossentropy (Stage 2)
- Batch Size: 32 (adjustable)
- Epochs: 50 (adjustable)
- Augmentation: Enabled by default
- Validation Split: 20%

## 🎯 Inference Pipeline

```
Input Image
    ↓
Preprocess (resize, normalize)
    ↓
Stage 1: Binary Detection
    ├─ If prob < 0.5 → "Not a mango"
    └─ If prob ≥ 0.5 → Continue
         ↓
    Stage 2: Variety Classifier
         ├─ Alphonso (prob)
         ├─ Kesar (prob)
         ├─ Langda (prob)
         └─ Other (prob)
         ↓
    Return: {variety, confidence}
```

## ⚙️ Customization

### Adjust Training
```yaml
# config.yaml
training:
  epochs: 100           # More epochs
  batch_size: 16        # Smaller batch (less memory)
  learning_rate: 0.001  # Faster learning
```

### Add New Varieties
1. Create directory: `data/raw/mango/new_variety/`
2. Update `src/models/mobilenet.py` → `MANGO_VARIETIES`
3. Retrain: `python main.py train`

